# Proyecto Sistemas Inteligentes - Sistema Recomendador

## Descripción

Este proyecto implementa un sistema recomendador utilizando técnicas de aprendizaje automático y procesamiento de datos. El objetivo es proporcionar recomendaciones personalizadas de películas basadas en las preferencias y comportamientos de los usuarios.
